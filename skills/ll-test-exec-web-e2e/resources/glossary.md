# Glossary

- **TESTSET**: The formal strategy-level QA object consumed by this skill.
- **TestEnvironmentSpec**: The structured execution environment contract referenced by `test_environment_ref`.
- **UI Intent**: The semantic step layer derived from governance-oriented test inputs before executable locators exist.
- **UI Source Context**: The normalized record of optional UI binding sources such as frontend code or runtime DOM.
- **UI Binding Map**: The resolution layer that maps semantic targets to executable Web locators or marks them partial.
- **TSE**: Test Set Execution, the execution-fact summary produced by the runtime.
- **Candidate Artifact**: The governed object registered before downstream gate review.
