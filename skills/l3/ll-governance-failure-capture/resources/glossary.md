# Glossary

- `failure_case`
  - governed failure record for `P0` and `P1`
- `issue_log`
  - light capture record for `P2`
- `diagnosis_stub`
  - initial classification only, not the final diagnosis
- `repair_context`
  - constrained context for later manual or human-directed AI repair
- `capture_manifest`
  - package index and file map
- `failure_scope`
  - one of `run`, `artifact`, or `field_cluster`
- `triage_level`
  - one of `P0`, `P1`, or `P2`
