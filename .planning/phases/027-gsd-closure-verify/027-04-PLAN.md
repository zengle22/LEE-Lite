# 027-04-PLAN: Integration tests for bug closure loop

## Goal
Create end-to-end integration test covering the full bug lifecycle.

## Requirements
- INTEG-TEST-01 (full loop test)

## Tasks

### Task 1: Create tests/integration/test_bug_closure.py
1. Set up temporary directory with test fixtures
2. Test the full loop:
   a. Create a test set and run qa-test-run to generate a detected bug
   b. Run gate-evaluate with FAIL to promote to open
   c. Run ll-bug-remediate to create phase
   d. Run /gsd-execute-phase to fix the bug
   e. Run qa-test-run --verify-bugs to verify and auto-close
3. Verify all state transitions are correct
4. Verify audit log has all entries

## Success Criteria
- [ ] Integration test passes
- [ ] All state transitions are verified
- [ ] Audit log is complete
