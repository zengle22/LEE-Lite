2. AnchorEntry frozen dataclass:
```python
@dataclass(frozen=True)
class AnchorEntry:
    anchor_id: str          # Format: PREFIX-NNN (e.g., JRN-001, ENT-001, SM-001)
    frz_ref: str            # FRZ-xxx reference
    projection_path: str    # One of: SRC, EPIC, FEAT
    metadata: dict[str, Any] = field(default_factory=dict)
    registered_at: str | None = None  # ISO 8601
```

3. ANCHOR_ID_PATTERN regex: `r"^[A-Z]{2,5}-\d{3,}$"` — uppercase prefix (2-5 chars), dash, 3+ digit number.

4. AnchorRegistry class:
   - Constructor: `__init__(self, workspace_root: Path)` — stores workspace_root, sets `registry_path = workspace_root / "ssot" / "registry" / "anchor_registry.yaml"`
   - `_load(self) -> list[dict[str, Any]]`: Read YAML file, return list of anchor records. Return empty list if file doesn't exist.
   - `_save(self, records: list[dict[str, Any]]) -> None`: Write records to YAML with key `anchor_registry`. Use `ensure_parent()` before write.
   - `register(self, anchor_id: str, frz_ref: str, projection_path: str, metadata: dict[str, Any] | None = None) -> AnchorEntry`:
     - Validate anchor_id matches ANCHOR_ID_PATTERN — raise `CommandError("INVALID_REQUEST", f"Invalid anchor ID format: {anchor_id}")` if not
     - Validate projection_path is one of: "SRC", "EPIC", "FEAT" — raise `CommandError("INVALID_REQUEST", ...)` if not
     - Check for duplicate anchor_id — raise `CommandError("INVALID_REQUEST", f"Anchor ID already registered: {anchor_id}")` if duplicate
     - Create record with `registered_at` as current UTC ISO 8601 timestamp
     - Append to registry, save, return AnchorEntry
   - `resolve(self, anchor_id: str) -> AnchorEntry | None`: Look up by anchor_id, return AnchorEntry or None
   - `list_by_frz(self, frz_ref: str) -> list[AnchorEntry]`: Filter records by frz_ref, return list of AnchorEntry
   - `list_all(self) -> list[AnchorEntry]`: Return all entries
   - `count(self) -> int`: Return number of registered anchors

5. Helper function: `def _dict_to_entry(record: dict[str, Any]) -> AnchorEntry:` — converts dict to AnchorEntry dataclass.

6. Follow existing `registry_store.py` patterns: use `yaml.safe_load`, `yaml.dump` with `default_flow_style=False`, `allow_unicode=True`, `sort_keys=False`.
