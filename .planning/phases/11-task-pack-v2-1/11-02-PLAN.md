```python
class TaskPackResolverError(ValueError):
    """Raised when dependency resolution fails (circular deps, missing refs)."""
```

3. `resolve_order(pack_yaml: dict) -> list[str]`:
   - Input: raw parsed YAML dict (the inner dict, with or without "task_pack" wrapper)
   - Extract tasks list: if "task_pack" key present, use data["task_pack"]["tasks"]; else use data["tasks"]
   - Collect all task_ids: `task_ids = {t["task_id"] for t in tasks}`
   - Validate all depends_on references exist:
     ```python
     for t in tasks:
         for dep in (t.get("depends_on") or []):
             if dep not in task_ids:
                 raise TaskPackResolverError(
                     f"Task '{t['task_id']}' depends on unknown task '{dep}'"
                 )
     ```
   - Build graph: `graph = {t["task_id"]: set(t.get("depends_on") or []) for t in tasks}`
   - Run topological sort:
     ```python
     try:
         ts = TopologicalSorter(graph)
         return list(ts.static_order())
     except CycleError as e:
         raise TaskPackResolverError(f"Circular dependency detected: {e}") from e
     ```

4. `resolve_file(path: str | Path) -> list[str]`:
   - Load YAML with `yaml.safe_load()`
   - Call `resolve_order(data)`
   - Return list of task_ids

5. `main()` — CLI entry point:
   ```python
   def main() -> None:
       """Resolve dependency order for one or more Task Pack YAML files.
       Usage: python -m cli.lib.task_pack_resolver <file.yaml> [...]
       """
       import sys
       args = sys.argv[1:]
       if not args:
           print("Usage: python -m cli.lib.task_pack_resolver <file.yaml> ...")
           sys.exit(1)
       exit_code = 0
       for f in args:
           try:
               order = resolve_file(f)
               print(f"OK: {f}")
               for i, tid in enumerate(order, 1):
                   print(f"  {i}. {tid}")
           except (TaskPackResolverError, FileNotFoundError) as e:
               print(f"FAIL: {f} — {e}")
               exit_code = 1
           except Exception as e:
               print(f"ERR : {f} — unexpected: {e}")
               exit_code = 1
       sys.exit(exit_code)
   ```

The module must NOT import the full schema module to avoid circular imports — only import `TaskPackSchemaError` and `validate` as needed. Use direct dict operations on the parsed YAML.

```python
def test_diamond_dependency():
    """A -> B, A -> C, B -> D, C -> D: diamond resolves correctly."""
    pack = {
        "tasks": [
```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| YAML file → resolver | Untrusted YAML files parsed — must use safe_load |
| depends_on references | Malicious or malformed references could cause resolver to process unexpected data |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-04 | Tampering | YAML parsing (`yaml.safe_load` in resolver) | mitigate | Use `yaml.safe_load()` exclusively — prevents YAML deserialization attacks (consistent with Plan 01) |
| T-11-05 | Denial of Service | Circular dependency graph | mitigate | `CycleError` from TopologicalSorter catches cycles immediately — no infinite loops possible; error message identifies the cycle |
| T-11-06 | Tampering | Orphan depends_on references | mitigate | All depends_on refs validated against known task_ids before graph construction — raises TaskPackResolverError for unknown refs |
| T-11-07 | Information Disclosure | Resolver error messages | accept | Error messages may expose internal task structure; low risk for local dev tool |
</threat_model>
