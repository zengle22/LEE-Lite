   class EnumGuardError(ValueError):

4. **EnumGuardError dataclass (frozen, for structured errors):**
   ```python
   @dataclass(frozen=True)
   class EnumGuardViolation:
       field: str            # e.g., "skill_id"
       value: str            # the invalid value
       allowed: list[str]    # allowed_values for this field
       label: str            # context (e.g., "module XYZ")

       def __str__(self) -> str:
           return (
               f"{self.label}: {self.field} must be one of {self.allowed}, "
               f"got '{self.value}'"
           )
   ```

8. **validate_field(field_name, value, label) -> list[EnumGuardViolation]:**
   - Look up field_name in ENUM_REGISTRY
   - If not found: return [EnumGuardViolation(field=field_name, value=value, allowed=[], label=label, ...)]
   - Get allowed values from enum class
   - If value not in allowed: return [EnumGuardViolation with field, value, allowed, label]
   - If valid: return []

9. **validate_enums(data: dict, label: str) -> list[EnumGuardViolation]:**
   - Iterate all keys in data
   - For each key in ENUM_REGISTRY, call validate_field()
   - Accumulate ALL violations (NOT fail-fast)
   - Return list of all EnumGuardViolation

10. **check_field(field_name, value, label) -> None:**
    - Convenience function that raises EnumGuardError on first violation
    - Calls validate_field(), if errors found, raise EnumGuardError with first error's str()

    - Parse --field, --check arguments
    - For --field: validate single field + value, print OK/FAIL
