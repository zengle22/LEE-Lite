## plan.yaml structures (ADR-047 §4.1.5 + §4.2.5)

### api_test_plan
```yaml
api_test_plan:
  feature_id: "feat.xxx"                        # string, required
  source_feat_refs: ["feat.xxx.section.1"]       # list[string], required
  api_objects:                                    # list[object], required
    - name: "training_plan"                       # string, required
      capabilities: ["create_plan", "get_plan"]   # list[string], required
  priorities:                                     # object, required
    p0: ["create_plan"]                           # list[string]
    p1: ["get_plan"]                              # list[string]
    p2: []                                        # list[string]
  cut_records:                                    # list[object], optional
    - capability: "update_plan"                   # string, required
      dimension: "幂等/重试/并发"                  # string, required
      cut_reason: "..."                           # string, required
      source_ref: "feat.xxx.rationale"            # string, required
```

### e2e_journey_plan
```yaml
e2e_journey_plan:
  prototype_id: "proto.xxx.v1"                   # string, required
  source_prototype_refs: ["proto.xxx.v1.flow.x"] # list[string], required
  journeys:                                       # list[object], required
    - journey_id: "journey.onboarding.create"     # string, required
      type: "main|branch|exception|revisit|state" # enum, required
      priority: "P0|P1|P2"                        # enum, required
  journey_cut_records:                            # list[object], optional
    - cut_target: "journey_type.exception"        # string, required
      cut_reason: "..."                           # string, required
      source_ref: "proto.xxx.page-type"           # string, required
      approver: "name"                            # string, required
      approved_at: "ISO8601"                      # string, required
```
