<action>
Create `cli/lib/gate_remediation.py`:

1. **Module docstring:** Reference ADR-055 §2.4, §2.8A.
2. **Imports:** from __future__ import annotations, uuid, datetime, Path, Any, yaml.
3. **Import from bug_registry:** load_or_create_registry, registry_path, transition_bug_status, _save_registry, _timestamp.
4. **Functions:**
   - `promote_detected_to_open(workspace_root: Path, feat_ref: str, gap_list: list[dict], run_id: str, actor: str = "system:gate-remediation") -> dict`
     - Loads registry, finds detected bugs where coverage_id in gap_list
     - Transitions each to open with reason="gate FAIL verdict"
     - Returns summary: promoted_count, already_open_count, archived_count
   - `archive_detected_not_in_gap_list(workspace_root: Path, feat_ref: str, gap_list: list[dict], run_id: str) -> dict`
     - Archives detected bugs NOT in gap_list (settlement says they're not real issues)
   - `_get_coverage_ids_from_gap_list(gap_list: list[dict]) -> set[str]`
     - Extracts coverage_id from each gap_list item
   - `_write_audit_log(workspace_root: Path, feat_ref: str, entries: list[dict]) -> None`
     - Writes audit entries to artifacts/bugs/{feat_ref}/audit.log (append-only)

5. **Audit log entry format:**
   ```yaml
   - timestamp: "2026-04-29T10:00:00Z"
     bug_id: BUG-xxx-abc123
     from: detected
     to: open
     actor: system:gate-remediation
     run_id: RUN-xxx
     reason: gate FAIL verdict
   ```
