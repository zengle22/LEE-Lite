   - `create_draft_phase_preview(workspace_root: Path, feat_ref: str, bugs: list[dict], run_id: str) -> Path`
     - Creates .planning/drafts/ directory if needed
     - Writes preview file: .planning/drafts/{timestamp}-bug-fix-preview-{feat_ref}.md
     - Preview includes bug list: bug_id, title, severity, gap_type, affected_files
   - `schedule_reminder(workspace_root: Path, feat_ref: str, bugs: list[dict], trigger_at: datetime, reminder_type: str = "t4h") -> None`
     - Writes reminder to artifacts/bugs/{feat_ref}/reminders.yaml (append-only)
     - Reminder entry: trigger_at, bug_count, reminder_type, acknowledged: false
   - `get_next_phase_number(workspace_root: Path) -> int`
     - Scans .planning/phases/ directories, extracts phase numbers, returns max+1
