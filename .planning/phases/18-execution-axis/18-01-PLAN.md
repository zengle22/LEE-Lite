4. Create `generate_run_manifest()` function:
   ```python
   def generate_run_manifest(
       workspace_root: Path,
       run_id: str,
       *,
       app_url: str,
       api_url: str | None = None,
       browser: str = "chromium",
       accounts: list[str] | None = None,
   ) -> Path:
   ```
   - Get git SHA via `subprocess.run(["git", "rev-parse", "HEAD"], capture_output=True, text=True).stdout.strip()`
   - Get frontend_build and backend_build (call `_get_build_version(workspace_root / "frontend")` and `... / "backend")`)
   - Create `_get_build_version()` helper that returns "unknown" if path doesn't exist or is not a git repo
   - Build manifest dict with fields:
     - `run_id`: str (e.g., "e2e.run-20260424-ABC12345")
     - `run_id_format`: "e2e.run-{timestamp}-{random}"
     - `created_at`: utc_now()
     - `git_sha`: str (40 char commit hash or "unknown")
     - `frontend_build`: str (version from frontend dir or "unknown")
     - `backend_build`: str (version from backend dir or "unknown")
     - `base_url`: dict with `app` and `api` keys
     - `browser`: str
     - `accounts`: list[str] (default empty list)
     - `artifact_version`: "1.0"
   - Create directory `ssot/tests/.artifacts/runs/{run_id}/`
   - Write manifest to `ssot/tests/.artifacts/runs/{run_id}/run-manifest.yaml`
   - Return the Path to the manifest file

5. Create `load_run_manifest()` function:
   ```python
   def load_run_manifest(workspace_root: Path, run_id: str) -> dict[str, Any]:
   ```
   - Load and return the manifest dict from `ssot/tests/.artifacts/runs/{run_id}/run-manifest.yaml`
   - Raise FileNotFoundError if manifest doesn't exist

6. Create `get_run_id_from_manifest_path()` helper:
   ```python
   def get_run_id_from_manifest_path(manifest_path: Path) -> str | None:
   ```
   - Extract run_id from manifest path (e.g., "e2e.run-20260424-ABC12345")
   - Return None if path is invalid

7. Create `list_run_manifests()` function:
   ```python
   def list_run_manifests(workspace_root: Path) -> list[str]:
   ```
   - List all run_ids in `ssot/tests/.artifacts/runs/`
   - Return sorted list of run_id strings
