6. Create dataclasses:
   ```python
   @dataclass
   class ALayerAssertion:
       """UI state assertions (visible elements, text, etc.)"""
       description: str
       source_step: str | None = None

   @dataclass
   class BLayerAssertion:
       """Network/API assertions with fallback handling"""
       description: str
       network_event: str | None = None
       fallback_available: bool = False
       fallback_description: str | None = None

   @dataclass
   class CLayerAssertion:
       """Business state assertions (marked C_MISSING)"""
       type: Literal["C_MISSING"] = "C_MISSING"
       description: str
       evidence_required: list[str] = field(default_factory=lambda: ["har", "screenshot"])
       placeholder: str = C_MISSING_PLACEHOLDER

   @dataclass
   class ScenarioStep:
       """Single step in a scenario"""
       step_index: int
       action: str
       target: str | None = None
       target_format: str | None = None
       value: str | None = None

   @dataclass
   class ScenarioAssertions:
       """All assertion layers for a scenario"""
       a_layer: list[ALayerAssertion] = field(default_factory=list)
       b_layer: list[BLayerAssertion] = field(default_factory=list)
       c_layer: list[CLayerAssertion] = field(default_factory=list)

   @dataclass
   class ScenarioSpec:
       """Compiled scenario spec with A/B/C layers"""
       journey_id: str
       spec_id: str
       coverage_id: str
       steps: list[ScenarioStep] = field(default_factory=list)
       assertions: ScenarioAssertions = field(default_factory=ScenarioAssertions)
       source_file: Path | None = None
   ```

7. Create `compile_scenario_spec()` function:
   ```python
   def compile_scenario_spec(e2e_spec: dict[str, Any], source_file: Path | None = None) -> ScenarioSpec:
   ```
   - Extract journey_id from `e2e_spec.get("_source_coverage_id", e2e_spec.get("unit_ref", ""))`
   - Extract spec_id from `e2e_spec.get("unit_ref", "")`
   - Extract coverage_id from `e2e_spec.get("_source_coverage_id", "")`
   - Build A-layer from `pass_conditions` in e2e_spec:
     - Filter conditions containing keywords: "visible", "显示", "show", "display", "text", "contain", "have"
     - Create ALayerAssertion for each
   - Build B-layer from `required_evidence` containing "network_log" or from `pass_conditions` with network keywords:
     - Keywords: "api", "network", "request", "response", "http", "/api/"
     - If no explicit network assertion found, add fallback BLayerAssertion
   - Build C-layer from `pass_conditions` with persistence keywords:
     - Keywords: "persistence", "persist", "database", "db", "backend", "server"
     - Create CLayerAssertion with type="C_MISSING"
     - Set evidence_required to ["har", "screenshot"] per D-04

8. Create `compile_e2e_spec_file()` function:
   ```python
   def compile_e2e_spec_file(spec_path: Path) -> ScenarioSpec:
   ```
   - Call `spec_parse_e2e_spec(spec_path)` from spec_adapter
   - Call `compile_scenario_spec()` with result
   - Set source_file to spec_path
   - Return ScenarioSpec

9. Create `compile_scenario_spec_batch()` function:
   ```python
   def compile_scenario_spec_batch(spec_paths: list[Path]) -> list[ScenarioSpec]:
   ```
   - Iterate over spec_paths and call `compile_e2e_spec_file()` for each
   - Return list of ScenarioSpec

10. Create `scenario_spec_to_yaml()` function:
    ```python
    def scenario_spec_to_yaml(spec: ScenarioSpec) -> str:
    ```
    - Convert ScenarioSpec to dict for YAML serialization
    - Handle dataclass fields properly
    - Return YAML string

11. Create `write_scenario_spec()` function:
    ```python
    def write_scenario_spec(spec: ScenarioSpec, output_dir: Path) -> Path:
    ```
    - Create output directory if needed
    - Write YAML to `{output_dir}/{spec.journey_id}.yaml`
    - Return Path to written file
