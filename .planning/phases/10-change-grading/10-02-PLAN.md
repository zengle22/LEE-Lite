**settle_minor_patch(patch_yaml, workspace_root) function:**
1. Extract grade_level from patch_yaml. If not present, call derive_grade(change_class) — import from patch_schema.py, do NOT re-implement.
2. **Idempotency check**: If patch status is already "applied", return early with {"status": "already_applied", "message": "Patch already settled"} — do not re-process.
3. If grade_level == "major", raise CommandError("INVALID_REQUEST", "Major patches must use ll-frz-manage --type revise, not settle")
4. Look up change_class in BACKWRITE_MAP
5. For each backwrite target:
   - Write backwrite RECORD to ssot/experience-patches/{feat_ref}/backwrites/{target}_updates.yaml
   - The record contains: patch_id, change_class, grade_level, changed_files, test_impact, timestamp
   - These are STRUCTURED RECORDS for human review, NOT actual SSOT file modifications.
6. Update patch_yaml["status"] = "applied"
7. Update patch_yaml["settled_at"] = current ISO timestamp
8. Write updated patch YAML back to its original location
9. Return dict: {"status": "applied", "backwrite_targets": [...], "files_written": [...]}
